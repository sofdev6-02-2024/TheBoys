## Instructions

1. Clone the repository

```bash
git clone https://github.com/sofdev6-02-2024/TheBoys.git
```

2. Install the dependencies

```bash
npm install
```

3. Build docker image

```bash
docker compose up 
```

This could take a while in first run, so be patient.
